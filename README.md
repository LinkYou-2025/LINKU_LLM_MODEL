# LINKU-LLM-MODEL

KoBART 기반 블로그 요약 서비스 API 서버

## Features

- KoBART 한국어 요약 모델
- FastAPI + Transformers
- Docker 배포 가능
- Spring Boot와 연동 가능

## Quick Start

```bash
git clone https://github.com/장서원/LINKU-LLM-MODEL.git
cd LINKU-LLM-MODEL
pip install -r requirements.txt
uvicorn app.main:app --reload
```
